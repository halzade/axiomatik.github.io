status popis vysvětlivky
100 Continue Server přijal úvodní část požadavku a klient může pokračovat.
101 Switching Protocols Server přepíná na protokol požadovaný klientem.
102 Processing Požadavek byl přijat a je zpracováván.
200 OK Požadavek byl úspěšně zpracován.
201 Created Byl úspěšně vytvořen nový zdroj.
202 Accepted Požadavek byl přijat ke zpracování, ale ještě nebyl dokončen.
204 No Content Požadavek byl úspěšný, ale odpověď neobsahuje žádná data.
206 Partial Content Server vrací pouze část požadovaného obsahu.
301 Moved Permanently Zdroj byl trvale přesunut na novou adresu.
302 Found Zdroj je dočasně dostupný na jiné adrese.
303 See Other Odpověď je dostupná na jiné URL pomocí metody GET.
304 Not Modified Zdroj nebyl změněn a lze použít verzi z cache.
400 Bad Request Požadavek je syntakticky chybný nebo neplatný.
401 Unauthorized Je vyžadováno ověření uživatele.
403 Forbidden Server požadavek pochopil, ale odmítá jej provést.
404 Not Found Požadovaný zdroj nebyl na serveru nalezen.
405 Method Not Allowed Použitá HTTP metoda není pro tento zdroj povolena.
408 Request Timeout Server neobdržel požadavek v očekávaném čase.
410 Gone Zdroj byl trvale odstraněn a již není dostupný.
429 Too Many Requests Byl překročen povolený počet požadavků.
500 Internal Server Error Na serveru došlo k interní chybě.
502 Bad Gateway Server obdržel neplatnou odpověď od nadřazeného serveru.
503 Service Unavailable Služba je momentálně nedostupná.
504 Gateway Timeout Nadřazený server neodpověděl včas.
505 HTTP Version Not Supported Server nepodporuje použitou verzi HTTP protokolu.
